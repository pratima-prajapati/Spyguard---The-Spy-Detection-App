# SpyGuard – The Spy Detection App

## About
An Android-based mobile security application developed to detect suspicious applications through permission-based heuristic analysis.

## Tech Stack
- Java
- Android Studio
- SQLite
- SMTP
- PackageManager API

## Features
- Real-time scanning of installed apps
- OTP-based authentication for secure login
- Threat alerts for suspicious applications
- Scan history storage using SQLite database
- User-friendly interface for mobile security awareness

## Purpose
Designed to enhance mobile security and protect user privacy by monitoring app permissions and detecting potential threats.
